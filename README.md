# DrawTrace AR

A browser-based camera-assisted tracing application that helps users transform reference images into hand-drawn sketches using real-time image overlays and device camera integration.

## Overview

DrawTrace AR allows artists, students, designers, and hobbyists to import images, overlay them on a live camera feed, and trace them directly onto paper. The application provides intuitive controls for opacity, zoom, rotation, image flipping, and outline visualization, creating a practical digital tracing experience without requiring specialized hardware.

## Features

* Camera-assisted tracing workspace
* Real-time image overlay system
* Adjustable image opacity
* Zoom and rotation controls
* Horizontal and vertical image flipping
* Outline mode for easier tracing
* Drag-and-drop image upload
* URL-based image import
* Mobile-friendly responsive interface
* Cross-platform browser compatibility

## How It Works

1. Upload an image from your device or provide an image URL.
2. Launch the camera workspace.
3. Overlay the image on the live camera feed.
4. Adjust opacity, scale, rotation, and orientation.
5. Position your device above paper and trace the image.
